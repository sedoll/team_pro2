# 📂 프로젝트 소개 
​
안녕하세요! **teenBook**은 '학생 및 교사 전용 커뮤니티 사이트입니다.  
이 곳은 서로의 경험, 관심사, 의견을 공유하며 함께 성장하고 배우는 공간을 제공합니다.
​
 학생들과 교사들이 협력하고 소통하며
 **서로의 경험을 공유하고 함께 성장할 수 있는 플랫폼을 제공**하는 것이   
 저희 팀의 기획의도 입니다.
